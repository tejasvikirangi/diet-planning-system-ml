# 🥗 Diet Planning System using BMI & Health Conditions

## 📌 Project Overview
This project is a Machine Learning-based web application that recommends personalized diet plans based on a user's BMI and health conditions.

The system analyzes user input and predicts suitable food recommendations using a trained ML model.

## 🚀 Features
- BMI calculation
- Health condition-based filtering
- Machine Learning prediction model
- User-friendly web interface (Flask)

## 🛠 Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- HTML/CSS
- Pickle (Model Deployment)

## ⚙️ How to Run the Project

1. Clone the repository
2. Install dependencies:

```
pip install -r requirements.txt
```

3. Run the application:

```
python app.py
```

4. Open browser and go to:
```
http://127.0.0.1:5000
```

## 📊 Machine Learning Model
The model was trained using supervised learning techniques on a structured food dataset.
